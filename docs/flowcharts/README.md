# Study Forge AI Flowcharts

This folder contains Mermaid flowcharts for core product flows.

## Files

- knowledge-pipeline.mmd
- quiz-pipeline.mmd
- interactive-chat.mmd

## Usage

1. Open any `.mmd` file.
2. Paste its content into a Mermaid-enabled viewer (for example, Mermaid Live Editor).
3. Export as SVG or PNG from the browser tool if needed.
